# Ethio Scholars Pathway

Ethio Scholars Pathway is a scholarship preparation platform designed to help talented Ethiopian high school students prepare for U.S. college admission and scholarship opportunities.

The platform provides academic preparation, mentorship, college application guidance, essay support, scholarship-readiness training, and donor-supported educational assistance.

## Mission

To identify talented Ethiopian high school students, prepare them for U.S. college admission, and connect them with scholarship opportunities through academic training, mentorship, and financial support.

## Important Note

Ethio Scholars Pathway does not guarantee admission, scholarships, visas, or travel to the United States. The program prepares students to compete strongly and honestly for available opportunities.

## First Pilot Goal

The first pilot program will support 20 Ethiopian high school students through a six-month online preparation program.

## Core Program Areas

- U.S. college application preparation
- Academic English and essay writing
- SAT, ACT, TOEFL, or Duolingo English Test preparation support
- Mentorship
- Scholarship search guidance
- Financial need review
- Donor-supported student assistance

## First Version Tools

The first version of this platform can be built using:

- Website: Wix or another no-code website builder
- Application form: Google Forms or Microsoft Forms
- Student database: Google Sheets or Airtable
- Online classes: Zoom or Google Meet
- Communication: WhatsApp, Telegram, and email
- Documents: Google Drive

## Future Platform Features

Later, this project can grow into a custom web application with:

- Student login
- Application dashboard
- Student testing system
- Mentor dashboard
- Scholarship tracker
- Document upload
- Donor dashboard
