# First 30-Day Launch Checklist

## Week 1: Foundation

- Choose final platform name
- Write mission and vision
- Choose slogan
- Create logo concept
- Buy domain name
- Create official email address
- Choose website builder, recommended first option: Wix

## Week 2: Forms and Selection System

- Build student application form
- Build teacher recommendation form
- Build parent consent form
- Create Google Sheet or Airtable database
- Create selection rubric
- Draft student privacy statement

## Week 3: Partnerships

- Contact Ethiopian schools
- Contact Ethiopian community leaders in the U.S.
- Contact possible mentors
- Contact college-advising organizations
- Prepare school nomination email

## Week 4: Open Pilot Applications

- Announce pilot program
- Collect applications
- Review applications
- Interview top students
- Select first 20 students
- Schedule orientation meeting

# Recommended First Platform Setup

Start simple:

- Wix website
- Google Forms for applications
- Google Sheets for student database
- Zoom or Google Meet for classes
- WhatsApp or Telegram for student communication
- Google Drive for documents

Later, build a custom web platform with student login, mentor dashboard, testing, document upload, and scholarship tracker.
